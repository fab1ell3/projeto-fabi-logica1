fabielle dos santos
tabuada
ele funciona da seguinte forma, escreva um número desejado para completar a tabuada do 1 até o 10, após disso o programa vai rodar e fazer os calculos da tabuada do número que você ordenou, como ele tem o programa de loop aplicado, o programa da tabuada só vai encerrar se digitar o número 0, ao contrário disso ele irá repitir.

tabela
o meu programa de tabela funciona como um sistema de cadastro de pessoas, contendo os espaços para colocar o nome, idade, altura, peso, sexo e número de telefone. o loop só sera interrompido quando o contador for = ou > que o número 5.

ordenação de números
esse programa vai perguntar quantos números voce quer que ele ordene, se digitar 3 por exemplo, ele ira te dar 3 espaços para escrever 3 números que quer, aleatório, após disso ele vai ordenar de forma crescente. Depois que ele ordenar o loop vai ser acionado e vai peguntar se deseja repetir, se sim escreva s, e se quiser encerrar o programa digite n.
